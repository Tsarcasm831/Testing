Swift Release (迅遁, Jinton, Viz: Quick Style) is a combined nature transformation kekkei genkai that can be used to move with incredible speed, becoming nothing more than a blur when in motion. Therefore, taijutsu becomes ineffective if the opponent is not fast enough.

Trivia

  • Swift Release shares the same romanisation as Dust Release.
  • A non-Konohagakure shinobi from one of the Five Great Shinobi Countries possessed this kekkei genkai.
  • In Naruto Mobile, Swift Release is implied to be created by simultaneously combining the Wind and Lightning natures.
